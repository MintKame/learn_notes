3. 二维数组

```java
int array[2][3] = {
    {0, 1, 2},
    {3, 4, 5}
};
// array[行][列]. 行指出哪个int[],列指出int[]中哪个int 
array[1][0] = 3
```

+ 二维数组在内存的地址： 不同语言的内存管理不一样
